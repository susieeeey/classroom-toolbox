课堂工具箱音频素材占位说明

当前版本已经接入统一音频管理器，并使用 WebAudio 生成柔和音效，不会主动加载 mp3 文件，因此线上部署不会因为缺少音频素材产生 404。
后续如需替换为真实素材，请将音频文件放入本文件夹，并在统一音频管理器中增加对应加载逻辑。

- click.mp3：普通按钮点击、转盘轻微咔哒声
- start.mp3：游戏开始提示
- suspense.mp3：随机点名、抽奖、抽签时的轻快悬念音乐
- ticking.mp3：倒计时、炸弹倒计时的轻微滴答声
- drum-loop.mp3：击鼓传花循环鼓点
- roulette-loop.mp3：幸运转盘旋转循环音乐
- card-flip.mp3：卡牌翻转、重新洗牌
- reveal.mp3：最终结果揭晓
- success.mp3：成功、完成、定格提示
- applause.mp3：颁奖、抽中后的掌声氛围
- countdown.mp3：最后 3 秒提示音
- explosion-soft.mp3：炸弹倒计时结束的柔和气球爆开音
- score-add.mp3：小组或个人加分
- score-minus.mp3：扣分的柔和提示
- background-light.mp3：可选的轻背景音乐
