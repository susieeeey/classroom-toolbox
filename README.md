# 小学课堂互动工具箱

正式访问网址：部署完成后填写

这是一个面向小学课堂的在线互动工具。老师打开网页即可使用，不需要登录，不需要安装软件，也不需要后台服务。

## 主要功能

- 学生名单粘贴导入、重复姓名提醒、请假/暂停参与管理
- 随机点名、不重复点名、幸运转盘、魔法抽签、幸运号码、幸运卡牌
- 翻牌挑战、翻牌抽人、随机题号、倒计时抢答、击鼓传花、炸弹倒计时
- 自动随机分组、小组 PK 积分、小组排行榜、小组抽人、随机组长、随机搭档
- 幸运之星、今日课堂 MVP、勇敢回答奖、进步之星、课堂积极之星
- 左下角课堂积分面板、积分排行榜、快捷加分、撤销和本节课重置

## 教师使用步骤

打开网址 → 粘贴名单 → 导入名单 → 选择游戏 → 开始互动

## 数据保存说明

学生名单、积分、点名记录、获奖记录和音效设置只保存在当前设备的浏览器 localStorage 中，不会上传到服务器。不同老师、不同设备、不同浏览器之间的数据互不共享。更换设备或清除浏览器数据后，原数据不会自动同步。

首页提供“一键清除本机数据”按钮，可清除当前浏览器中保存的名单、积分、记录和设置。

## 音效启用说明

浏览器会限制自动播放声音。首次打开网页后，请点击“试听”或任意课堂游戏按钮启用课堂音效。Safari 和手机浏览器需要先有一次点击或触摸操作，之后游戏音乐和提示音会正常播放。

当前版本使用 Web Audio API 生成课堂音效，不依赖外部音频文件。`assets/audio/` 目录保留给后续替换真实音频素材使用。

## 如何更新网页

修改 `index.html`、`manifest.json`、`service-worker.js` 或资源文件后，提交代码并重新部署。正式网址会保持不变。

## 如何重新部署

本项目是静态网页，`index.html` 位于部署根目录。推荐使用 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel。部署时确保根目录包含：

```text
index.html
manifest.json
service-worker.js
assets/
README.md
.nojekyll
```

GitHub Pages 推荐设置：

1. 新建 GitHub 仓库。
2. 上传本项目根目录文件到 `main` 分支。
3. 打开仓库 `Settings` → `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`。
6. 保存后等待 1 到 3 分钟生成公开网址。

## 常见问题

问：老师的数据会不会被其他人看到？

答：不会。学生名单和积分只保存在当前浏览器本机，不上传服务器。

问：为什么第一次没有声音？

答：浏览器限制自动播放，请先点击页面按钮或“试听”启用音效。

问：手机可以用吗？

答：可以。页面已做响应式适配，手机端积分面板会以底部抽屉方式显示。

问：翻牌抽人翻开后看不到姓名怎么办？

答：当前版本已在卡牌正面和页面中间结果区同时显示姓名，Safari 也保留了 3D 翻转兼容样式。
